# 3D Models

3D printable case designs for the Conan-style voice changer project. All models are designed with [Autodesk Fusion](https://www.autodesk.com/products/fusion-360/).

## Bowtie Microphone Case

A classic red bowtie-shaped case housing the BM83 Bluetooth module, MEMS microphone, and battery.

### Assembly Preview

![Bowtie Mic Assembly](https://github.com/CreatorKanata/conan-style-voice-changer-mic-speaker/blob/main/3d-models/bowtie-mic/red-bowtie-mic-asm.jpg?raw=true)

### Parts

| File | Description |
|------|-------------|
| `red-bowtie-mic.step` | Top shell (red bowtie shape) |
| `red-bowtie-mic-bottom.step` | Bottom cover |

### Components Inside

- BM83 Bluetooth module PCB
- MEMS microphone
- LiPo battery
- Power switch

---

## Button Speaker Case

A compact circular button-shaped case for the speaker unit.

### Assembly Preview

![Button Speaker Assembly](https://github.com/CreatorKanata/conan-style-voice-changer-mic-speaker/blob/main/3d-models/button-speaker/button-speaker-asm.jpg?raw=true)

### Parts

| File | Description |
|------|-------------|
| `button-speaker-top.step` | Top cover with speaker grill |
| `button-speaker-bottom.step` | Bottom shell |

### Components Inside

- BM83 Bluetooth module PCB
- Thin speaker driver
- LiPo battery
- Power switch

---

## Printing Recommendations

| Parameter | Recommendation |
|-----------|----------------|
| Material | PLA or PETG |
| Layer Height | 0.2mm |
| Infill | 20-30% |
| Supports | May be needed for overhangs |
