# PCB Designs

Custom PCB designs for the Conan-style voice changer project. All boards are designed with [EasyEDA Pro](https://pro.easyeda.com/) for easy ordering from JLCPCB.

## Bowtie Microphone

Bluetooth microphone board with BM83 module and MEMS microphone.

### Features

- BM83 Bluetooth module (HFP profile)
- MEMS microphone for high-quality voice capture
- Battery charging circuit
- Compact form factor to fit in bowtie case

- Compact circular design for button case

## Manufacturing

All PCBs are designed for manufacturing at [JLCPCB](https://jlcpcb.com).

### Ordering Steps

1. Go to [JLCPCB](https://jlcpcb.com)
2. Upload the Gerber ZIP file
3. Select PCB options (color, thickness, etc.)
4. Enable SMT Assembly if needed
5. Upload BOM and Pick-and-Place files for assembly
6. Place order
